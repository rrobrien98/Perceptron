How to execute this project.
Simply run the main method of Lab.java on the command line with the following parameters in order
1:Number of desired input nodes in perceptron, either 1024 or 64
2:Number of desired output nodes in perceptron, either 10 or 1
3:Filename of file containing train data. NOTE, WE ERASED ALL HEADER INFORMATION IN OUR TEST FILES TO MAKE PARSING SIMPLER. ie the first line in all the testfiles was already classification info
4:Number of iterations to train the network for
5:Learning rate
6:Number of tests to run through network
7:Filename of file containing test data. Same warning applies as for train data

The main function will print out how many inputs were classified correctly
